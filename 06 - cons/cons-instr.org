* Objetivo

Estimar una función de consumo agregado para el área formada por los
países que inicialmente formaron el área Euro utilizando datos
trimestrales de los años 1977--1998.

* Datos

Se utilizan parte de la base de datos que compiló el Banco Central
Europeo para estimar el Area Wide Model, un modelo econométrico que se
usó para modelizar los principales agregados macroeconómicos del área
formada por los países que constituyeron el área Euro. El modelo es
descrito en [cite:@faganAreawideModelAWM2001].
